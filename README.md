# Movie Recommendation App

## CI/CD Assignment Report – Movie Recommendation App

### 1. Project Title
Movie Recommendation App using GitHub CI/CD

### 2. Objective
To automate the build and deployment of a movie recommendation web application using GitHub Actions and GitHub Pages, implementing a Continuous Integration and Continuous Deployment (CI/CD) workflow.

### 3. Tools & Technologies

| Tool | Purpose |
|------|---------|
| Git | Version control for source code |
| GitHub | Repository hosting and collaboration |
| GitHub Actions | Continuous Integration and Deployment automation |
| GitHub Pages | Hosting the static website |
| Node.js + Jest | Testing JavaScript functionality |

### 4. Workflow Steps (CI/CD Pipeline)

| Stage | Description | Tool Used |
|-------|-------------|-----------|
| Code Commit | Developer pushes code to GitHub repository (main branch) | Git |
| Build | GitHub Actions automatically triggers on every push and installs dependencies | GitHub Actions |
| Test | Runs automated Jest test cases to ensure app functionality | Jest |
| Deploy | If all tests pass, GitHub Pages automatically deploys the updated code to a live link | GitHub Pages |

### 7. Advantages of CI/CD

- **Automated testing** ensures code quality
- **Rapid deployment** of new features and fixes
- **Continuous feedback** for developers
- **Reduced manual errors** in deployment process
- **Faster time to market** for applications
- **Consistent deployment** across environments

### 8. Features

- Search for movies by title
- Display detailed movie information
- Get movie recommendations based on genres
- Responsive design for all devices
- Real-time search functionality
- Powered by OMDb API
